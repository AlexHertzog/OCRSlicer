OCRSlicer iOS 
=================

- No OS X support.
- Strict requirement on language files existing in a referenced "tessdata" folder.

Instalation
=================

Using cocoa pods just add the following line to your podfile
```pod 'OCRSlicer'```

and run: 
```pod install```
